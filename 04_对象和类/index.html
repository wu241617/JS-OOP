JS 对象的种类

  • JS内置的对象（如Number）
  • 宿主环境对象（如window）
  • 自己创建的对象

JS 对象的创建

  • 对象字面量形式（属性，方法，get set型属性）
  • 成员访问：两种访问方式（. []）。访问对象中没有的属性时，会返回undefined。级联访问（对象里面嵌套对象）
  • Object 函数创建，Object.defineProperty定义与修改属性。Object.defineProperties定义和修改多个属性。还可以用o.xxx='xxx'或者o['xxx']='xxx'去新增属性
  • 对象工厂方式（构造函数），创建对象

JS 对象的基本操作

  • 成员的遍历: for(el in o),反射性遍历，无序。可以使用Object.keys(o)辅助排序，进行有序操作。
  • 检查对象是否有某个值: 'xxx' in o 或者 o.hasOwnProperty('xxx')
  • 成员的删除:delete o.'xxx' ,有些属性是不能删除的

JS 对象成员特性

  • writable:规定是否可改写
  • enuerable:规定是否可枚举（决定遍历时是否出现）
  • configurable:规定是否可配置
  • 通过 Object.getOwnPropertyDescriptor(o,'xxx') 得到自己属性的描述器

constructor 构造器

  • 类型判断：typeof 和 instanceof
